# Contributing

Thanks for your interest in research-orchestrator!

This project is in early development. Before starting work on a pull request, please **open an Issue first** to discuss what you'd like to change. This helps avoid duplicated effort and ensures your contribution fits the project direction.

## Setup

```bash
git clone <repo-url>
cd research-orchestrator
pip install -e ".[dev]"
pytest
```

## Pull Requests

- Keep PRs focused and small when possible
- Add tests for new functionality
- Make sure `pytest` passes before submitting

## Bug Reports

Please include:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Python version and OS

## Questions?

Feel free to open an Issue with the "question" label.
